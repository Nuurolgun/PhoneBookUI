# ag‑Grid Template Component

This component provides a ready‑to‑use ag‑Grid example with a few sample columns and rows. It is a **standalone** Angular component that can be dropped into any template.

## Files
- `ag-grid-template.component.ts` – component class with default `columnDefs` and `rowData`.
- `ag-grid-template.component.html` – `<ag-grid-angular>` markup.
- `ag-grid-template.component.scss` – basic styling.

## Usage
```html
<app-ag-grid-template></app-ag-grid-template>
```
Add the selector to any template (e.g., `app.component.html`). The component imports `AgGridAngular` and `CommonModule` automatically.

## Customisation
If you need custom columns or data, you can modify the `columnDefs` and `rowData` properties in the TypeScript file, or extend the component to accept `@Input()` values.

## Styling
The component uses the **Alpine** theme by default (`class="ag-theme-alpine"`). You can change the theme by updating the class in the HTML file.
